Euler's Totient Function

May 22, 2022 • By Sham Number Theory

Introduction

For \( n\geq 1 \), \( \phi(n) \) denote the number of positive integers not exceeding n that are relatively prime to \( n \).

For example, \( \phi(30)=8 \) as the numbers which are relatively prime to \( n \) are \( 1, 7, 11, 13, 17, 19, 23, 29 \).

Properties

  • If \( p \) is prime, \( \phi(p)=p-1 \).
  • \( \phi(n) \) is always even for \( n > 2 \).
  • \( \phi(mn) = \phi(m)\phi(n) \) if \( \gcd(m,n)=1 \).
  • Gauss: \( n = \sum_{d|n} \phi(d) \).

Computation

A simple \( O(\sqrt{n}) \) algorithm implementation in C++:

int phi(int n) {
    int result = n;
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) {
            while (n % i == 0)
                n /= i;
            result -= result / i;
        }
    }
    if (n > 1)
        result -= result / n;
    return result;
}
