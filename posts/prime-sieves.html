Prime Sieves: Sieve of Eratosthenes

September 6, 2020 • By Sham Code Primes

I'm writing a series on different sieve methods for extracting primes (as a self project) below a given number. In this the first would be the sieve of Eratosthenes naturally as it is one of the simplest. Below is the code it writes out all primes below that number in .txt file. The number of operations are reduced as we are only calculating from the square of the prime to \( \sqrt{n} \).

import time
import math

def eratosthenes_sieve(b):
    primes = [True,]*b

    for j in range(2, math.ceil(math.sqrt(b))):
        if primes[j]:
            for i in range(j*j, b, j):
                    primes[i] = False

    return primes

The Sieve of Eratosthenes is one of the most efficient ways to find all prime numbers up to a specified integer.
